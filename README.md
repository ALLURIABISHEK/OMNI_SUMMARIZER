## 🌟 **What is Omni Summarizer?**

**Omni Summarizer** is a cutting-edge full-stack web application that revolutionizes content consumption by providing **intelligent summarization across multiple formats**. Built with state-of-the-art AI models, it transforms how users interact with text, documents, images, and videos.

### ✨ **Key Highlights**
- 🚀 **Multi-Modal Processing** - Handle text, PDFs, images, and videos seamlessly
- 🤖 **Advanced AI Integration** - Powered by Gemini 1.5 Flash, BLIP-2, ViT-GPT, GIT, and CLIP
- ⚡ **Real-Time Processing** - Lightning-fast summarization with performance tracking
- 📱 **Responsive Design** - Works flawlessly across all devices
- 🔄 **Model Comparison** - Side-by-side AI model performance analysis

## 🚀 **Getting Started**

### Prerequisites
- Python 3.8+
- MySQL Server
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/omni-summarizer.git
   cd omni-summarizer
   ```

2. **Set up virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure database**
   ```bash
   # Create MySQL database and update config
   mysql -u root -p
   CREATE DATABASE omni_summarizer;
   ```

5. **Set environment variables**
   ```bash
   export GEMINI_API_KEY="your_api_key_here"
   export DATABASE_URL="mysql://user:password@localhost/omni_summarizer"
   ```

6. **Run the application**
   ```bash
   python app.py
   ```
